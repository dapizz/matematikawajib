Persamaan dan Pertidaksamaan

Memahami persamaan linear, kuadrat, pertidaksamaan, serta sistem persamaan linear dua dan tiga variabel.

Materi Inti

1. Persamaan Linear Satu Variabel (PLSV)

Persamaan linear satu variabel adalah persamaan yang mempunyai satu variabel dengan pangkat satu. Bentuk umumnya adalah:

Bentuk Umum PLSV
ax + b = 0, dengan a ≠ 0

Langkah Penyelesaian:

  1. Pindahkan konstanta ke ruas kanan
  2. Pindahkan variabel ke ruas kiri
  3. Sederhanakan kedua ruas
  4. Bagi dengan koefisien variabel

2. Persamaan Kuadrat

Persamaan kuadrat adalah persamaan dengan variabel berpangkat dua. Bentuk umumnya:

Bentuk Umum Persamaan Kuadrat
ax² + bx + c = 0, dengan a ≠ 0

Metode 1: Faktorisasi

Ubah persamaan menjadi bentuk perkalian faktor-faktor:

Contoh Faktorisasi
x² - 5x + 6 = 0
(x - 2)(x - 3) = 0
x = 2 atau x = 3

Metode 2: Rumus Kuadrat (ABC Formula)

Gunakan rumus ketika faktorisasi sulit dilakukan:

Rumus Kuadrat
x = -b ± √(b² - 4ac) 2a

Diskriminan (D = b² - 4ac):

  • D > 0: Dua akar berbeda
  • D = 0: Dua akar kembar
  • D < 0: Tidak memiliki akar real

3. Pertidaksamaan Linear

Pertidaksamaan linear menggunakan tanda <, >, ≤, atau ≥. Penyelesaian berupa interval nilai.

Penting!

Jika mengalikan atau membagi dengan bilangan negatif, tanda pertidaksamaan harus dibalik.

Contoh Penyelesaian:

2x - 4 > 6

x = 5

2x > 10 → x > 5

4. Pertidaksamaan Kuadrat

Langkah penyelesaian:

  1. Ubah ke bentuk standar (kanan = 0)
  2. Carikan akar-akar persamaan kuadrat
  3. Buat garis bilangan dan tentukan daerah penyelesaian

Aturan Tanda:

  • Untuk a > 0 (parabola terbuka ke atas): positif di luar akar
  • Untuk a < 0 (parabola terbuka ke bawah): positif di dalam akar

5. Sistem Persamaan Linear Dua Variabel (SPLDV)

SPLDV adalah dua persamaan linear dengan dua variabel yang harus diselesaikan bersama-sama.

Bentuk Umum SPLDV
a₁x + b₁y = c₁
a₂x + b₂y = c₂

Metode Substitusi:

  1. Ubah salah satu persamaan menjadi x = ... atau y = ...
  2. Substitusikan ke persamaan lainnya
  3. Selesaikan persamaan satu variabel
  4. Substitusikan kembali untuk mencari variabel lain

Metode Eliminasi:

  1. Samakan koefisien salah satu variabel
  2. Eliminasi dengan menjumlahkan atau mengurangkan
  3. Selesaikan variabel yang tersisa
  4. Substitusikan untuk mencari variabel lain

6. Sistem Persamaan Linear Tiga Variabel (SPLTV)

SPLTV adalah sistem tiga persamaan dengan tiga variabel. Penyelesaian dengan eliminasi bertahap.

Bentuk Umum SPLTV
a₁x + b₁y + c₁z = d₁
a₂x + b₂y + c₂z = d₂
a₃x + b₃y + c₃z = d₃

Langkah Penyelesaian:

  1. Eliminasi satu variabel dari dua pasang persamaan
  2. Hasilnya adalah SPLDV
  3. Selesaikan SPLDV tersebut
  4. Substitusikan untuk mencari variabel yang dieliminasi

Animasi Interaktif

Grafik Persamaan Linear: y = mx + c
Gradien (m): 1
Konstanta (c): 0
Gradien (m) menentukan kemiringan garis. Konstanta (c) menentukan titik potong dengan sumbu-y.
Grafik Persamaan Kuadrat: y = ax² + bx + c
Koefisien a: 1
Koefisien b: 0
Koefisien c: 0
Nilai a menentukan arah dan kelengkungan parabola. Akar-akar ditandai dengan titik pada sumbu-x.
Daerah Pertidaksamaan Linear
Gradien (m): 1
Daerah yang diarsir menunjukkan himpunan penyelesaian pertidaksamaan. Garis putus-putus untuk tanda < atau >.

Contoh Soal dan Pembahasan

1 Persamaan Linear Satu Variabel

Selesaikan: 3(2x - 4) = 5x + 2

1
Langkah 1

Kurung dibuka dengan distribusi

6x - 12 = 5x + 2
2
Langkah 2

Pindahkan variabel ke kiri, konstanta ke kanan

6x - 5x = 2 + 12
3
Langkah 3

Sederhanakan

x = 14
Jawaban: x = 14
2 Persamaan Kuadrat - Faktorisasi

Selesaikan: x² - 7x + 12 = 0

1
Langkah 1

Cari dua bilangan yang hasil kalinya 12 dan jumlahnya -7

Bilangan: -3 dan -4 (karena -3 × -4 = 12 dan -3 + -4 = -7)
2
Langkah 2

Faktorisasi

(x - 3)(x - 4) = 0
3
Langkah 3

Tentukan akar-akar

x - 3 = 0 → x = 3 x - 4 = 0 → x = 4
Jawaban: x = 3 atau x = 4
3 Persamaan Kuadrat - Rumus Kuadrat

Selesaikan: 2x² + 5x - 3 = 0

1
Langkah 1

Identifikasi nilai a, b, c

a = 2, b = 5, c = -3
2
Langkah 2

Hitung diskriminan

D = b² - 4ac = 25 - 4(2)(-3) = 25 + 24 = 49
3
Langkah 3

Substitusi ke rumus kuadrat

x = -5 ± √49 2(2) = -5 ± 7 4
4
Langkah 4

Hitung kedua nilai

x₁ = -5 + 7 4 = 2 4 = 1 2 x₂ = -5 - 7 4 = -12 4 = -3
Jawaban: x = 12 atau x = -3
4 Pertidaksamaan Kuadrat

Selesaikan: x² - x - 6 < 0

1
Langkah 1

Cari akar-akar persamaan x² - x - 6 = 0

(x - 3)(x + 2) = 0 → x = 3 atau x = -2
2
Langkah 2

Buat garis bilangan dan uji tanda

-2
3
3
Langkah 3

Karena a = 1 > 0 (parabola terbuka ke atas) dan kita cari yang < 0, maka ambil daerah di antara akar-akar

Jawaban: -2 < x < 3
5 SPLDV - Metode Eliminasi

Selesaikan sistem persamaan:

2x + 3y = 12 ... (1)
3x - 2y = 5 ... (2)

1
Langkah 1

Eliminasi y: kalikan persamaan (1) dengan 2 dan persamaan (2) dengan 3

4x + 6y = 24 ... (3) 9x - 6y = 15 ... (4)
2
Langkah 2

Jumlahkan (3) dan (4)

13x = 39 → x = 3
3
Langkah 3

Substitusi x = 3 ke persamaan (1)

2(3) + 3y = 12 6 + 3y = 12 3y = 6 → y = 2
Jawaban: x = 3, y = 2

Latihan Soal

1

Selesaikan: 4x - 7 = 2x + 9

Hint: Pindahkan variabel ke satu ruas

2

Selesaikan dengan faktorisasi: x² + 8x + 15 = 0

Hint: Cari dua bilangan yang hasil kalinya 15 dan jumlahnya 8

3

Selesaikan: -2x + 5 > 11

Hint: Ingat! Bagi dengan bilangan negatif = balik tanda

4

Tentukan penyelesaian: x² - 4x + 3 ≥ 0

Hint: Gunakan garis bilangan setelah menemukan akar-akar

5

Selesaikan SPLDV:
x + y = 7
2x - y = 2

Hint: Coba gunakan metode eliminasi
